# Security Policy

This project is a governance framework for AI-assisted recursive improvement.
It should not contain secrets, credentials, private prompts, private logs, or
deployment keys.

## Reportable Issues

Please report concerns involving:

- secret leakage
- unsafe autonomous behavior
- prompt or tool-injection surfaces
- bypassable veto or approval gates
- misleading promotion records

## Public Boundary

Public examples should remain small, reproducible, and domain-neutral. Do not
publish private operational logs, credentials, or data that identifies users,
customers, systems, or internal workflows.
